# 性能优化

五大优化维度

1. 连接优化
  - CDN
  - DNS预解析
  - 减少重定向
  - 减少并发（tcp阻塞、http1.1队头阻塞，采用多个域名、http2）

2. 缓存优化
  - http缓存
    - 强缓存
    - 协商缓存
  - 客户端缓存
    - PWA、App Cache
    - 初始数据缓存

3. 资源优化
  - 按需加载
    - 分包
    - 动态导入
    - 图片、组件懒加载
  - 压缩
    - gzip、br
    - 尺寸、格式（webp、webm...）
    - 代码压缩 uglifyjs 、 esbuild

4. 代码优化
  - html + css
    - 节点数量
    - 不同步加载 `<script>`，放到后面或者 `defer` （文档解析完后，最后执行）、`async`（加载完js立马执行）
    - 减少重排、重绘
  - js
    - 代码质量
    - 逻辑后移（后端）

5. 体验优化
  - 动画
  - 反馈
  - loading
