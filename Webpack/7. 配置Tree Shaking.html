# 配置Tree Shaking

Tree Shaking 只支持 ES Module，如：import { add } from '@/utils/index' 不支持 CommonJS，如：const utils = require('@/utils/index');

# 对 "sideEffect": false 浅显的解释

会在 Babel 转换后执行判断能否“摇掉”部分没有引入的代码，但是会通过语法分析，保留有“副作用”的代码（包括原本没副作用，Babel 转换后导致有副作用的代码）。
